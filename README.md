# javascript-calculator-precourse

## 구현할 기능 목록

1. 핵심 기능: 더하기 함수

- [x] 매개변수를 숫자로 이루어진 배열로 받는다.
- [x] 초기 값은 0이다.
- [x] 배열을 순회하면서 숫자를 더해 나간다.
- [x] 총 합계를 반환한다.

2. 분리 기능

- [ ] 구분자를 쉼표(,) 또는 콜론(:)으로 초기 세팅한다.
- [ ] 커스텀 구분자인 경우
  - [ ] 구분자를 찾아서 업데이트한다.
  - [ ] 커스텀 구분자 표기를 기준으로 입력값을 업데이트한다.
- [ ] 구분자를 기준으로 분리한다.
- [ ] 분리된 문자열을 숫자로 변환한다.
